# Instalación

Para instalar la herramienta, siga estos pasos:

1. Clonar el repositorio:
   ```bash
   git clone https://github.com/usuario/gestion-de-ramas-funcionalidad.git
   cd gestion-de-ramas-funcionalidad
   ```
2. Instalar las dependencias:
   ```bash
   npm install
   ```
3. Ejecutar la aplicación:
   ```bash
   npm start
   ```

Asegúrese de tener instalado Node.js (versión 16 o superior) y Git en su sistema.
