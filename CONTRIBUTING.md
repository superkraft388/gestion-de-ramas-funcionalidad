# Contribuciones

¡Gracias por considerar contribuir a este proyecto!

## Cómo Contribuir
1. Haga un fork del repositorio.
2. Cree una rama para su característica (`git checkout -b mi-feature`).
3. Realice sus cambios y haga un commit (`git commit -am 'Agregar nueva característica'`).
4. Envíe su rama (`git push origin mi-feature`).
5. Abra un Pull Request.

Por favor, asegúrese de que su código sigue los estándares de estilo del proyecto, que las pruebas sean correctas y que se han ejecutado antes de enviar el Pull Request.
